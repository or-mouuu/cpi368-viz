# 「持續上漲」類別細緻化分類建議

目前「持續上漲」在您的分類體系中是最大的區塊（n=155）。為了解決分類不夠細緻的問題，我們對這些品項的「12個月移動平均趨勢線」進行了幾何特徵運算（包含線性擬合度、前後期斜率對比、最大跳躍幅度等），建議可依據數據折線圖的形狀，將「持續上漲」進一步細分為以下四種不同的「漲相」：

## 1. 晚期加速型 (Accelerating Growth) —— 佔比最大
- **形狀特徵**：十年的前段價格相對平穩或緩慢爬升，但最近幾年（尤其是疫情後）趨勢線斜率明顯變陡，通膨壓力在近期才劇烈發酵。
- **數據條件參考**：後半段區間的斜率顯著大於前半段（例如 `slope2 > slope1 * 2`）。雖然它們可能不滿足您原先設定的「近期急漲（近三年貢獻比 ≥ 55%）」那樣極端的門檻，但仍具有明顯的尾盤加速特徵。
- **代表品項**：豬肉、牛肉、米粉、蛋糕、其他調理食品等。
- **故事意義**：這類商品是民眾近期對於「物價變貴」最有痛感的來源，前期溫水煮青蛙，後期漲勢凌厲。

## 2. 穩定長線型 (Linear/Constant Growth) —— 第二大宗
- **形狀特徵**：十年間呈現近乎完美的直線遞增，沒有經歷暴漲也沒有停滯，就像爬樓梯一樣穩定地慢慢變貴。
- **數據條件參考**：趨勢線與線性迴歸直線高度擬合（$R^2 > 0.8$ 或 $0.9$），且前後期斜率差異不大。
- **代表品項**：速食麵、麵包、羊肉、火腿臘肉等。
- **故事意義**：這類商品的定價較不受短期原物料或突發事件的劇烈衝擊，而是反映長期的人工與營運成本上升，長年下來無聲無息地吃掉購買力。

## 3. 階梯跳漲型 (Step-like Growth)
- **形狀特徵**：長時間（可能長達數年）價格維持一條水平線完全不動，但偶爾會出現一次極度陡峭的「跳躍式」調漲，調完價後又繼續維持長期的平坦。
- **數據條件參考**：最大單月漲幅遠大於平均單月變化（高 Jumpiness），且整體線性擬合度較低。
- **代表品項**：香菸（通常受菸稅政策影響）、國內外旅遊團費、部分個人照顧服務費、皮蛋等。
- **故事意義**：這類商品多半具有「僵固性定價」模式，例如廠商久久才敢漲一次價（一漲就漲足），或是受政策稅制調整所驅動。

## 4. 早期上漲型 / 漲勢趨緩型 (Decelerating Growth)
- **形狀特徵**：十年間的早期曾經歷過一段漲幅，但近幾年價格逐漸趨於平緩，斜率明顯下降。
- **數據條件參考**：前半段區間的斜率顯著大於後半段。它們近期可能還有一點極微弱的漲幅，所以沒有被歸類到您既有的「漲後不跌（近12個月動能 < 1.5%）」中，但性格上已經接近高原期。
- **代表品項**：童上衣、釋迦等（在此類群中數量最少）。
- **故事意義**：歷史上曾漲過，但並非近年這波通膨的主力。

### 💡 實作建議
如果您希望在 UI/資料上實作這套細分類，可以考慮以下做法：
1. **新增次分類標籤**：在既有的「持續上漲」主分類下，為品項加上這四種標籤（如：`accelerating`, `linear`, `steplike`, `decelerating`），並在前端折線圖旁加上對應的小圖示（例如斜線、曲線、階梯圖示）。
2. **門檻微調**：可以透過計算各品項前 5 年與後 5 年的斜率比值（Acceleration Ratio）以及階梯跳躍係數（Jumpiness），來找出最適合您的資料分布的切分點。其中「晚期加速型」與「穩定長線型」是此區塊中最重要的兩個對比敘事。
