# 走勢分群分析（2026-07-11）

`shape_metrics.csv`：358 項各自的走勢指標與 k-means 分群結果。

指標定義（以 12 個月移動平均為趨勢線，排除季節噪音）：
- `total_change`：十年趨勢漲幅 %
- `gain_at_peak`：漲到最高點時的漲幅 %
- `drawdown`：現在距峰值回落 %
- `retrace`：漲上去的部分吐回了多少（0=站穩，1=全吐回，>1=跌破起點）
- `peak_pos`：峰值出現時點（0=十年前，1=現在）
- `volatility`：對趨勢線的偏離度（季節/波動性）
- `momentum`：近 12 個月趨勢變化 %
- `linearity`：趨勢線的線性度 R²

k=5 分群（silhouette 0.42）+ 最大群再切三刀，自然浮現約 6 種走勢型態；
結論與「漲相」性格對映討論見 ROADMAP.md「待重新思考」一節。
